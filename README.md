### LMS-UI

LMS Front End

#### Getting Started

1. Clone this repository

    ```bash
    git clone https://dev.azure.com/igetitcodin/IGETITV2/_git/UI LMS_UI
    ```

2. Go to project directory and install all dependencies

    ```bash
    cd LMS_UI
    npm install --legacy-peer-deps
    ```

3. Available scripts

    ```bash
    # Run development server
    npm run dev

    # Build project
    npm run build

    # Serve project
    npm run preview
    ```
